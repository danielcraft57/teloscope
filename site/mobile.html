Teloscope Mobile

Vérifiez un numéro, enregistrez un message pour les appelants commerciaux, et coupez ces appels sur Android.

Accueil Vérifier un numéro App mobile Filtre matériel À propos Contact Mentions légales

Fonctions

  • Vérification OSINT — même logique que le site, branchable sur VocalGuard.
  • Message enregistré — fichier audio associé aux blocages.
  • Filtrage Android — rôle système « filtrage d'appels », journal des coupes.
  • iPhone — vérification + message ; blocage via le boîtier matériel.

Code source
